Doormouse self service network registration

Doormouse is a device registration portal that your users can use to register their devices for use on your network. Doormouse means your customers get quick self service network access, whilst you get control over your network in line with compliance requirements.

What does it do

  • It authenticates users to establish identity
  • It tracks agreement with network access terms and conditions
  • It allows end user registration of devices
  • It can auto detect end user devices to aid registration process
  • It allows the end user to manage their own devices.
  • It allows network administrators to manage customer devices.
  • It keeps network access records as required for compliance purposes

Features

  • It can auto detect device mac addresses to aid device registration
  • It has native support for SAML2 identity infrastructure so it doesn't require the end user to have yet another password.
  • It has a rich permission system that allow you to accurately control who can access what
  • It has close help desk system integration so end users can get help easily
  • It has a comprehensive API so you can integrate your own systems with doormouse easily
  • It has a web interface styling system based on Thymeleaf so you have complete control on the appearance of your doormouse instance.
  • It can integrate with Microsoft DHCP
  • It has a command line interface
