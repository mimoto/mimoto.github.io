Doormouse: self service network registration

Doormouse is a device registration portal that your users can use to register their devices for use on your network. Doormouse means your customers get quick self service network access, whilst you get control over your network and compliance with legal requirements.

Benefits

  • Authenticates users to establish identity
  • Tracks consent to terms and conditions
  • Allows end-user registration of devices
  • Can auto-detect end-user devices to aid registration process
  • Allows the end-user to manage their own devices
  • Allows network administrators to manage customer devices
  • Keeps network access records as required for compliance purposes
  • More than one organisation can provide access to the same buildings

Features

  • Can auto-detect device MAC addresses to aid device registration
  • Has native support for SAML2 identity infrastructure so it doesn't require the end user to have yet another password
  • A rich permission system that allow you to accurately control who can access what
  • Close helpdesk system integration so end-users can get help easily
  • A comprehensive API so you can integrate your own systems with Doormouse
  • A web UI styling system based on Thymeleaf so you have complete control on the appearance of your Doormouse instance
  • Integrates with Microsoft DHCP and enterprise network hardware
  • Commandline interface for administration
